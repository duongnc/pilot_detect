# pilot_detect

Computer vision project for multimodal emotion detection.

Quick start

1. Create a Python virtual environment and install dependencies:

```bash
python -m venv env
source env/bin/activate
pip install -r requirements.txt
```

2. Run training or main scripts as needed (see project files).

Pushing to GitHub

This repository can be created on GitHub with the `gh` CLI or by creating a remote manually.

Example (with `gh` installed and authenticated):

```bash
gh repo create pilot_detect --public --source=. --remote=origin --push --confirm
```
